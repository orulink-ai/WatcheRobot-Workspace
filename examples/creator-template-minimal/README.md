# Creator Template Minimal

Use this folder as the starting point for community examples.

## Recommended Structure

```text
creator-template-minimal/
  README.md
  manifest.example.json
  assets/
```

## Manifest

See `manifest.example.json`.

## Rules

- Include a license for your assets.
- State the target firmware / app / desktop version.
- Include test evidence.
- Do not include private keys, API tokens, Wi-Fi credentials, or private user data.
