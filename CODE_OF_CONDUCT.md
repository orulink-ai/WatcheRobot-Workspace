# Code of Conduct

WatcheRobot aims to be a respectful, practical, and welcoming open-source robotics community.

## Expected Behavior

- Be respectful and specific.
- Assume good intent, but correct problems clearly.
- Keep discussions focused on the project.
- Share enough context for others to reproduce issues.
- Credit creators for examples, assets, hardware modifications, and tutorials.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing private keys, tokens, passwords, personal data, or private contact information.
- Misrepresenting unfinished features as finished.
- Pressuring maintainers for unpaid private support.

## Reporting

Security issues should follow `SECURITY.md`. Community conduct issues can be reported through the project maintainer channel once confirmed. Until the official community channel is confirmed, use GitHub Issues only for project-work topics and avoid posting private information.
