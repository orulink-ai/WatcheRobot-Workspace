# Open Source Scope

This document describes what is currently intended as public open-source material. Unknowns must remain as TODO/TBD.

## Currently Present in the Workspace

- Mobile app source: `WatcheRobot_app`
- Desktop client source: `WatcheRobot_client`
- Server source and docs: `WatcheRobot_server`
- ESP32-S3 firmware source and docs: `WatcheRobot_esp32`
- STM32 firmware source and docs: `WatcheRobot_stm32`
- Root workspace scripts and documentation
- Local examples being prepared under `examples/`

See [Hardware and Structure Map](hardware-structure-map.md) for the current resource inventory and approval status.

## Needs Confirmation Before Public Launch

- Final license strategy for root, desktop, server, and STM32.
- Which 3D / CAD / structural files may be published.
- Which product images, GIFs, videos, and brand assets are approved.
- Which community channel is official.
- Whether release binaries and checksums should be linked from the root README.

## Not Public Unless Confirmed

- Private API keys, signing keys, tokens, certificates, and local config.
- Personal COM ports, local paths, logs, caches, and machine-specific data.
- Unapproved production assets or unreleased mechanical files.
