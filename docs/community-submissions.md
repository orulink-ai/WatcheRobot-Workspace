# Community Submissions

WatcheRobot should make community contributions easy to review and credit.

## Accepted Submission Types

- Documentation fixes
- BLE examples
- Motion examples
- Expression / animation packs
- Hardware notes and assembly improvements
- Desktop or App UX improvements
- Tutorials and showcase entries

## Required Submission Info

- What changed
- Why it helps
- Files or assets included
- How it was tested
- License / ownership of any submitted asset
- Whether the submission depends on a specific firmware, App, or desktop version

## Asset Rule

Do not submit assets you do not own or cannot license.
