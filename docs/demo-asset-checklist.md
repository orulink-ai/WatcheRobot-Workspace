## Approval Checklist

- [ ] Shows real WatcheRobot product, desktop UI, App UI, or verified hardware behavior.
- [ ] No private API keys, Wi-Fi names, faces, personal files, or private workspace content.
- [ ] No unapproved customer/user content.
- [ ] File has a clear source and license/usage approval.
- [ ] Asset is stored under `docs/assets/` or linked from an approved public URL.
- [ ] README first viewport still loads quickly.

## Naming Convention

Use lowercase, hyphenated names:

- `watche-robot-hero.jpg`
- `watche-robot-desktop-demo.gif`
- `watche-robot-app-ble-flow.png`

Do not copy assets into `docs/assets/` until approval is confirmed.
